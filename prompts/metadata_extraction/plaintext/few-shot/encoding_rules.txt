# General Guidelines

- Focus on accuracy over guessing
- Only resolve abbreviations when highly confident
- Include a confidence score from 0.0 (uncertain) to 1.0 (certain)
- Use "unk" (unknown) when information is unavailable or unclear

# Normalization Rules

## Date Format:
- Use W3C format: YYYY-MM-DD when day is known; YYYY-MM when only month is known; YYYY when only year is known
- If Julian and Gregorian dates are present in the letter, use the Gregorian date
- If not extractable from letter: unk

## Place Format:
- Normalize to contemporary English spelling (e.g., "Vienna" not "Wien", "Constantinopel" not "Istanbul")
- Always append ISO 3166-1 Alpha-2 two-letter country code: "Vienna | AT"
- If not extractable from letter: unk

## Name Format (Structured):
- Extract name components separately: forename, lastname, particles, profession
- **Forename**: First/given name (e.g., "Joseph", "Karl August")
  - Normalize to modern spelling (e.g., "Karl" not "Carl")
  - Resolve abbreviations (e.g., "J. H." to "Joseph") only when confident (conf ≥ 0.7); otherwise use "unk".
- **Lastname**: Family name (e.g., "Hammer-Purgstall", "Böttiger")
  - Resolve abbreviations (e.g., "J. H." to "Hammer-Purgstall") only when confident (conf ≥ 0.7); otherwise use "unk".
  - If unknown: unk
- **Particles**: Nobility particles (e.g., "von", "de", "van")
  - Omit if none or unknown
- **Profession**: Occupation or profession if found in letter or known
  - Use broad GND categories in German (e.g., "Orientalist", "Schriftsteller", "Marineoffizier")
  - Omit if none or unknown
- **Titles/ranks**: Exclude from all fields (e.g., omit "Freiherr", "Hofrat", "General")

## Language Format:
- Specify the primary language using ISO 639-1 two-letter code (e.g., "de", "fr", "en")
- If unknown: unk


# Confidence Scores

- 1.0: Explicitly named in dateline or signature + independently verifiable
- 0.9: Full name/location + strong contextual evidence
- 0.7-0.8: Full name/location + moderate evidence, OR abbreviation + strong evidence
- 0.5-0.6: Abbreviation or missing name component + moderate evidence
- <0.5: Weak evidence, but contextually plausible - flagged for human review

## Evidence levels:

Strong: Named explicitly in letter body, or externally verifiable
Moderate: Implied by context (address, signature style, known correspondents)
